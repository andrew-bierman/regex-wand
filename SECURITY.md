# Security Policy

## Supported Versions

Security fixes target the latest published `regex-wand` version.

## Reporting A Vulnerability

Please report security issues privately instead of opening a public issue.

Use GitHub's private vulnerability reporting for this repository when available.
If private reporting is not enabled yet, contact the maintainer directly through
GitHub.

Include:

- affected package version
- reproduction steps or proof of concept
- expected impact
- any known mitigations

Do not publish exploit details until a fix or mitigation is available.
